# 总部营销周会汇报

本 reference 根据 `D:\working files\宏洋总部2026-\营销周例会` 下 2026 年 1-4 月营销周例会 docx 提炼，用于形成总部周会、领导会、周度经营复盘口径。

## 汇报目的

周会不是项目流水账，而是让领导在短时间内判断四件事：

1. 本月指标能否完成，销售额和合约额分别能到哪里。
2. 变化来自哪里：市场、客户、竞品、项目节点、货量结构、团队执行。
3. 哪些项目是主力贡献，哪些项目是风险拖累，哪些项目需要领导支持。
4. 下周或节点前准备做什么，目标是多少，卡点是什么。

## 标准结构

### 地区层面

使用这个顺序：

1. 地区公司名称。
2. 月度指标、本月已售、认筹、在手金额、预计销售额、预计合约额。
3. 整体判断：来访、成交、客户、竞品、节点、项目结构的核心变化。
4. 完成判断：确保多少、冲刺多少、缺口多少、依赖哪些项目。
5. 关键风险或机会：五一、清明、元宵、返乡、回城、交付、取证、开盘、政策等节点。

表达示例结构：

> 月度指标 X，本月已售 X，认筹 X，在手 X，预计销售额 X，预计合约额 X。整体看，市场/来访/成交呈 X 趋势，主力贡献来自 X，风险集中在 X。按当前在手和节点动作，预计可确保 X，冲刺 X。

### 项目层面

按“客户-竞品-项目-分析-计划”写，但不要机械堆满。领导需要的是矛盾和动作。

1. 客户：本周来访、成交、认筹、新访/复访、户型结构、同比/环比、本月指标、已完成、目标销售/合约。
2. 竞品：价格、成交、来访、佣金、赠送、特价房、加推、降价、展示节点，以及对我方客户预期和流速的影响。
3. 项目：取证、样板间、展示区、交付、开盘、加推、工程、合作方、政策兑现等节点。
4. 分析：一句话说明核心矛盾，不要重复客户和竞品数据。
5. 计划：下周动作、目标金额/套数、打法抓手、需支持事项。

项目段落压缩模板：

> 项目 X：本周来访 X，成交 X，认筹 X，环比/同比 X；本月已完成 X，预计 X。成交集中在 X 户型/房源，当前核心问题是 X。竞品 X 通过 X 动作影响客户预期。下周围绕 X 节点，主推 X 房源，目标 X，需支持 X。

## 取舍规则

周会材料要区分“必须汇报”和“可以略过”：

- 主力项目：讲贡献、可持续性、后续货量和是否能继续冲。
- 风险项目：讲缺口、原因、解决动作和需要决策。
- 节点项目：讲开盘、加推、取证、展示、交付、政策窗口。
- 稳态小项目：只保留关键数字和一句判断，避免占用篇幅。
- 已完成项目：关注是否还有合约、认筹、尾款、价格体系和下月断档风险。

## 常见数据现象的解释

- 来访上升、成交下降：多为客户观望、价格预期不匹配、竞品更强、产品总价偏高，重点看复访和认筹。
- 来访下降、成交持平：可能是前期客储转化或节点逼定有效，但后续客储要预警。
- 来访和成交双降：优先排查市场热度、天气/假期、竞品动作、渠道停滞、展示和货量结构。
- 来访平稳、成交提升：说明转化动作、特价房、渠道聚焦、节点包装或房源挤压有效。
- 指标完成但流速下降：可能是拔价、优质房源减少、节点过后自然回落，要提示下月续航。
- 认筹高、成交低：关注取证、征信、价格、关系户、开盘节奏和解筹转化。
- 成交靠解筹或前期认筹：汇报时要说明“新增成交”和“解筹转化”差异，避免误判热度。

## 常见业务判断

- 春节后、元宵后、清明后、五一前后，要重点看返乡/回城置业是否真实转化，而不是只看来访。
- 节点政策要能延续热度：例如春节政策延续到元宵后，五一提前购用于月末逼定。
- “客户等政策”会压制短期来访和成交，要用提前购、限时名额、特价房、房源稀缺性做挤压。
- 竞品降价、加佣、送车位、送家电会直接拉低客户心理预期，必须判断我方是跟、绕开，还是强化价值。
- 对主力项目，要关注流速是否远超竞品、优质房源是否减少、是否需要抢窗口出货。
- 对尾盘和难点项目，要关注具体房源结构，不要只说市场差。

## 修改与压缩方法

如果用户提供“原始内容”，要改成更像总部周会的“修改内容”：

- 删除重复的同比/环比解释，只保留能支撑判断的数据。
- 把“项目、竞品、分析、计划”中重复的信息合并成一段。
- 对领导不需要听的过程性细节降级，只保留节点、影响和结论。
- 每个项目保留一个核心矛盾和一个下周动作。
- 地区层面必须有预计销售额/合约额和完成判断。

## 会前准备清单

- 本月指标、已售、认筹、在手、预计销售额、预计合约额。
- 周度来访、成交、认筹及同比/环比。
- 主力项目本周成交户型、面积段、房源位置和客户来源。
- 竞品最新价格、成交、加推、分销、赠送、特价房。
- 节点：开盘、加推、取证、样板间、售楼处、交付、政策截止。
- 本周需要领导拍板或跨部门协调的事项。

## 口径要求

- 讲趋势时必须带原因，讲原因时必须带动作。
- 不要把地区原话直接搬上会，要转成总部能判断的经营语言。
- 对“预计完成”要有依据：在手认筹、可转化客户、节点房源、政策动作、历史流速。
- 对“冲刺目标”要说明依赖条件：价格、渠道、取证、展示、财务、团队执行。
