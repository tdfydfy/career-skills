# 销售话术教练 Skill 使用说明

## 这个 Skill 是做什么的

`sales-talk-coach` 是一个房地产销售话术训练工具，分为两类功能：

1. 项目说辞考核：检查销售讲项目、讲沙盘、讲户型、讲园林、讲竞品时，是否覆盖了项目价值点和合规口径。
2. 客户接待对练：模拟真实客户，让销售在对话里练习转化、逼定和跟进。

它不是单纯帮销售写一句回复，而是既能考项目掌握度，也能练真实接待能力。

客户接待对练会训练：

- 怎么自然破冰
- 怎么挖掘客户真实需求
- 怎么识别预算、首付、月供和决策链
- 怎么把项目价值讲到客户关心的点上
- 怎么处理价格、竞品、学区、家人不同意、市场观望等异议
- 怎么合规推进下一步

它适合销售培训、晨夕会练习、新人带教、老销售复盘、案场话术统一和项目开盘前演练。

## 进入后先选模式

如果没有明确说明，建议先这样启动：

```text
使用 $sales-talk-coach
```

系统会先给出简短开场白，说明流程和用法，再询问你的需求：

```text
欢迎使用销售话术教练。
我可以帮你做两类训练：一类是项目说辞考核，检查你讲项目时价值点是否讲全、讲准、讲合规；另一类是客户接待对练，我会模拟真实客户，你来接待，结束后我会给评分、问题清单和优秀销售示范。
如果你没有提供真实项目资料，我会默认使用模拟项目“澜庭序”训练。

请先选择本次训练方式，回复 1 或 2 即可，也可以直接补充要求：
1. 项目说辞考核：你来讲项目/沙盘/价值点，我按项目资料检查覆盖率和合规性。
2. 客户接待对练：我扮演客户，你按真实接待来沟通，结束后复盘转化能力。
```

你可以直接回复：

```text
1
```

或：

```text
2
```

也可以补充要求：

```text
1，考核建筑园林和好房子标准
```

```text
2，难度高级，重点练价格异议
```

也可以直接指定模式。

```text
使用 $sales-talk-coach，做项目说辞考核，我来讲澜庭序整体沙盘。
```

```text
使用 $sales-talk-coach，做客户接待对练，难度中级。
```

## 核心机制

### 0. 体验模式和正式模式

如果你只是想看流程，可以说：

```text
体验一下项目说辞考核流程
```

体验模式里可以说“假设我已经回答”“下一题”，系统会继续演示流程。

正式考核或正式对练中不允许这样跳过。系统只会根据你实际回答的内容评分；如果你跳过，系统会要求你实际作答，或者把本次标记为“体验流程/考核未完整”。

### 1. 项目说辞考核

项目说辞考核不扮演客户，模型会以销售经理或教练的身份提问，重点检查销售是否真正掌握项目，而不是让销售一口气背完整说辞。

适合考核：

- 整体沙盘讲解
- 建筑园林说辞
- 苗木和景观价值
- 中建好房子标准
- 户型产品矩阵
- 价格付款口径
- 竞品对比口径
- 分客群价值表达

评分更看重“价值点覆盖率”。如果销售把给定项目资料中的核心价值点都讲到了，并且没有明显违规，就可以给高分。

项目说辞考核会严格按当前项目资料评分。你没有提供真实项目资料时，默认按“澜庭序”评分；如果你讲了与澜庭序资料不一致的内容，系统会标注“与项目资料不符/需确认”，不会算作已覆盖价值点。

### 2. 隐藏客户卡片

每次对练开始前，系统会随机生成一张客户卡片，但不会告诉销售人员。卡片包含：

- 家庭结构：年龄、父母、子女、当前居住状态
- 购房直接原因：学区、改善、结婚、养老、投资、交通等
- 购房深层原因：安全感、面子、夫妻关系、子女陪伴、父母照护等
- 资金情况：首付、月供、公积金、贷款资格、父母支持、夫妻资金博弈
- 决策链：谁出钱、谁住、谁拍板、谁黑脸、谁白脸
- 具体需求：户型、面积、楼层、景观、价格、装修、交付时间
- 来访信息：首访/复访、几个人来、是否看过竞品
- 聊天状态：了解、心动、犹豫、放弃

销售必须通过自然对话逐步挖掘这些信息。

客户接待对练开始时，系统会先给一段“现场可观察信息”，帮助你像真实案场一样进入状态。这部分只包括销售能直接看到或登记台能知道的表层信息，例如：

- 本次是首访还是复访。
- 来了几个人，大致性别和年龄。
- 穿着、气质、精神状态和随身物品。
- 到访时间是上午、下午还是晚上，是工作日还是周末。
- 当前事件，例如刚进售楼部、刚看完竞品、带孩子到访、复访直接问房号等。

系统不会在开场告诉你客户预算、真实动机、谁能拍板、谁出资、家庭矛盾或真正卡点。这些仍然需要你在对话中挖掘。

### 3. 默认模拟项目

如果没有提供真实项目资料，系统默认使用虚构项目“澜庭序”训练。该项目包含完整资料：

- 项目定位、区位交通、配套、产品矩阵
- 89/108/128/143 平户型价值点
- 建筑设计、园林理念、名贵苗木、中建好房子口径
- 价格区间、付款口径、竞品矩阵
- 常见异议和标准话术
- 首访、复访、沙盘、样板间、逼定流程

所以即使不配置项目，也可以直接开始训练。

### 4. 复盘评分

对练结束后，系统会输出结构化评估，包括：

- 本轮客户卡片摘要
- 总分和分项评分
- 信息获取完整度
- 项目价值传递质量
- 合规风险
- 主要问题和遗漏字段
- 改进话术
- 优秀销售示范片段
- 下一轮训练建议

### 5. 优秀销售示范

除了打分，系统还会演示优秀销售应该怎么说。示范不是“强行成交”，而是展示销售如何一步步：

1. 破冰
2. 挖需
3. 确认资金
4. 识别决策链
5. 复述需求
6. 转化项目价值
7. 处理异议
8. 锁定下一步

每句话后都会说明“这一句的目的”，方便新人理解好在哪里。

## 常见使用场景

### 场景一：项目说辞考核

适合检查销售是否掌握项目资料，而不是考察转化逼定。

示例指令：

```text
使用 $sales-talk-coach，做项目说辞考核，我来讲澜庭序整体沙盘。
```

```text
使用 $sales-talk-coach，考核我的建筑园林说辞，重点看价值点覆盖。
```

```text
使用 $sales-talk-coach，考核我讲中建好房子标准，讲完后给我评分。
```

```text
使用 $sales-talk-coach，考核户型产品说辞，我讲 89、108、128、143 平。
```

评分重点：

- 项目价值点是否覆盖完整
- 是否把价值点讲成客户利益
- 结构是否清楚
- 口径是否准确合规
- 是否有可复制的标准话术

### 场景二：新人销售日常对练

适合新人熟悉接待流程、客户提问和基本合规边界。

示例指令：

```text
使用 $sales-talk-coach，开始一轮新人销售对练，难度初级。
```

```text
使用 $sales-talk-coach，模拟一个首访客户，我来练开场和需求挖掘。
```

### 场景三：价格异议训练

适合训练预算、首付、月供、优惠申请和阶梯谈判。

示例指令：

```text
使用 $sales-talk-coach，重点训练价格异议，客户要比较难缠一点。
```

```text
使用 $sales-talk-coach，模拟一个预算卡得很紧但又喜欢项目的客户。
```

### 场景四：竞品对比训练

适合训练销售如何客观承认竞品优势，并把客户拉回需求排序。

示例指令：

```text
使用 $sales-talk-coach，模拟客户刚看完竞品江岸云邸，现在来对比澜庭序。
```

```text
使用 $sales-talk-coach，训练竞品对比，不允许贬低竞品。
```

### 场景五：家庭决策训练

适合训练夫妻、父母、子女、亲友共同参与时的决策链识别。

示例指令：

```text
使用 $sales-talk-coach，模拟夫妻复访，一个人心动，一个人黑脸压价。
```

```text
使用 $sales-talk-coach，模拟客户带父母来看房，我练怎么识别谁拍板。
```

### 场景六：高级价值点训练

适合训练建筑、园林、苗木、好房子标准等高级价值表达。

示例指令：

```text
使用 $sales-talk-coach，训练我怎么讲澜庭序的园林、苗木和建筑设计。
```

```text
使用 $sales-talk-coach，模拟一个客户质疑“好房子是不是包装概念”。
```

### 场景七：逼定和复访推进训练

适合训练如何锁复访、锁决策人、锁预算核算、锁房源对比。

示例指令：

```text
使用 $sales-talk-coach，模拟一个已经二访但还在犹豫的客户，我练逼定推进。
```

```text
使用 $sales-talk-coach，客户准备回去商量，我练怎么锁下一次复访。
```

### 场景八：只看优秀销售示范

如果不想先对练，也可以直接看标杆打法。

示例指令：

```text
使用 $sales-talk-coach，给我演示一轮优秀销售如何接待夫妻复访客户。
```

```text
使用 $sales-talk-coach，优秀销售会怎么处理“价格高、还要看竞品”的客户？
```

## 项目说辞考核怎么跑

### 第一步：启动

```text
使用 $sales-talk-coach，做项目说辞考核，主题是整体讲盘。
```

也可以指定专项：

```text
使用 $sales-talk-coach，做项目说辞考核，主题是建筑园林和苗木。
```

### 第二步：教练提问，销售回答

系统会先问一个概述题：

```text
请你先用 2 分钟概括一下澜庭序这个项目最核心的 3 到 5 个优势。先不用讲太细，我会根据你说的内容继续追问。
```

销售回答后，系统会根据你已讲和漏讲的内容继续追问，例如：

- 园林分区和苗木价值没有讲透
- 好房子标准没有转成客户痛点
- 户型产品没有对应家庭阶段
- 竞品对比缺少客观取舍
- 合规边界讲得过满

### 第三步：多轮追问

项目说辞价值点很多，系统不会三两轮就草草评分。默认会按模块逐项追问，通常覆盖 6 到 8 个专项模块后再建议评分。

常见追问模块包括：

- 项目定位
- 区位配套
- 建筑理念
- 园林与苗木
- 好房子标准
- 户型产品矩阵
- 价格付款
- 竞品对比
- 合规边界

你也可以随时说：

```text
讲完了，请评分。
```

如果中途评分，系统会标注“考核未完整”，并列出未考模块。

系统会输出：

- 总分
- 价值点覆盖率
- 已覆盖价值点
- 遗漏价值点
- 讲错或讲虚的地方
- 合规风险
- 优秀项目说辞示范

## 客户接待对练怎么跑

### 第一步：启动

```text
使用 $sales-talk-coach，做客户接待对练，难度中级。
```

系统会说明本轮使用默认模拟项目，先给出现场可观察信息，再以客户身份说第一句话。示例：

```text
已进入客户接待对练。本轮按默认项目澜庭序，我将扮演客户，你直接回复我即可。

现场可观察信息：
- 来访次数：首访。
- 来访人数：2 人，一男一女，30 岁上下。
- 穿着与状态：周末下午到访，穿休闲装，女方看起来比较有兴趣，男方话少。
- 当前事件：两人刚从竞品项目过来，进门后先停在区位图前。

客户：你们这个项目现在还有 100 平左右的吗？我们刚在网上看过一点。
```

### 第二步：销售正常对话

销售只需要像真实接待一样回复客户。不要问系统“客户是什么类型”，因为客户设定是隐藏的。

建议销售重点练习：

- 少背项目，多问客户
- 少急着报价，多拆首付和月供
- 少说卖点，多转成生活场景
- 少逼定，多锁一个下一步动作

### 第三步：结束复盘

当你想结束时，输入：

```text
结束，给我复盘评分。
```

或：

```text
复盘，并给我优秀销售示范。
```

系统会停止扮演客户，输出评估报告和标杆示范。

## 难度设置

可以在启动时指定难度。

```text
使用 $sales-talk-coach，开始一轮初级对练。
```

初级：客户相对配合，适合新人。

```text
使用 $sales-talk-coach，开始一轮中级对练。
```

中级：客户会有保留、比较、反问，适合日常训练。

```text
使用 $sales-talk-coach，开始一轮高级对练。
```

高级：客户多重异议交织，可能有黑白脸、预算博弈、竞品比较和合规陷阱。

## 使用真实项目资料

如果要用真实项目训练，可以先提供项目资料。建议包含：

- 项目名称、城市、板块、定位
- 主力户型、面积、总价、付款方式
- 核心卖点和真实短板
- 竞品项目及优劣势
- 学区、价格、优惠、交付等合规边界
- 常见异议和案场标准口径

示例指令：

```text
使用 $sales-talk-coach，下面是我们项目资料，请基于这个项目做一轮销售对练：
项目名称：……
主力户型：……
价格区间：……
核心卖点：……
竞品：……
合规边界：……
```

如果真实项目资料不完整，系统会只使用已确认信息，不会替你编造事实。

## 两类评分维度

### 项目说辞考核

总分 100 分，主要看五项：

- 价值点覆盖完整度：40 分
- 价值转化能力：20 分
- 结构与表达清晰度：15 分
- 准确性与合规性：15 分
- 记忆点与可复制性：10 分

这类考核更像“讲盘考试”。销售只要把项目资料中的核心价值点讲全、讲准、讲成客户能理解的利益，并且不违规，就能拿高分。

### 客户接待对练

总分 100 分，主要看六项：

- 开场与关系建立：10 分
- 信息获取完整度：25 分
- 需求诊断与客户理解：15 分
- 项目价值传递：20 分
- 异议处理与推进动作：15 分
- 合规与专业边界：15 分

高分销售的特点不是话术华丽，而是：

- 问得自然
- 听得准确
- 判断客户真实卡点
- 把项目价值讲到客户利益上
- 守住合规边界
- 每次沟通都能锁一个明确下一步

## 合规提醒

训练中不要使用以下说法：

- 保证学区、保证入学、绝对对口
- 一定升值、肯定不跌
- 最低价、内部价、领导特批只给你
- 明天一定涨价、马上没房
- 竞品有硬伤但没有证据
- 绕过家人先定
- 绝对不漏、不吵、不裂、零甲醛

正确做法是承认不确定性，用事实、证据链、取舍逻辑和下一步动作推进。

## 推荐训练节奏

新人销售：

- 每天 1 次初级对练
- 每次只练一个动作：开场、挖需、预算、竞品或复访
- 连续 5 天后再进入中级

成熟销售：

- 每周 2 到 3 次中级或高级对练
- 重点练复杂客户卡片、黑白脸、竞品和逼定
- 复盘后沉淀优秀话术

销售管理者：

- 晨会抽 1 个场景做 10 分钟演示
- 夕会复盘 1 个真实客户，转成模拟卡片再练
- 把高分示范话术沉淀到项目话术库
