# <项目名> 项目知识地图

## 文档用途

- 本文档是本项目知识库的入口地图，用来说明项目资料分布、优先阅读顺序和当前待补充项。
- 当用户提出一般性项目问题时，优先参考本地图，再按需进入对应主题文档。

## 使用规则

- 只把已填写、已确认的内容作为正式知识使用。
- 带有“待补充 / 待确认”的字段，不得直接对客户做事实性承诺。
- 学校、学区、招生、政策、价格折扣等高风险信息，仍需结合全局合规规则和最新项目口径。

## 推荐阅读顺序

1. `01-project-core-card`：项目总卡，快速了解位置、产品、客户、卖点与风险。
2. `02-product-and-pricing`：户型、面积、价格、付款和优惠。
3. `03-location-school-faq`：区位、配套、交通、学校和生活半径。
4. `04-competitor-summary`：竞品对比、板块选择、客户适配判断。
5. `05-objection-handling`：客户抗性、犹豫点、追问处理。
6. `06-project-dynamics`：近期销售节点、加推、优惠、动态变化。
