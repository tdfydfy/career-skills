# 案例：滨江一号 · 赣州

> 完整故事线创作流程，从破冰摸底到三件套交付。

---

## 项目信息

| 项 | 内容 |
|----|------|
| 位置 | 赣州章江新区，西侧临章江 |
| 布局 | 西南叠拼洋房（200㎡+） → 东北高层（140㎡+精装），西低东高户户看江 |
| 立面 | 新古典主义，帝国大厦同款风格 |
| 定位 | 高端改善，高出均价30% |
| 阶段 | 设计阶段 |
| 客群 | 40-50岁赣州人，在外打拼（广东/稀土/家具/电商）回乡置业，人生最后一套房 |
| 资源 | 一线临江+滨江公园+章江新区核心最后几块好地 |
| 竞品 | 有江景盘但定位偏低，无成熟故事线 |
| 案名 | 滨江一号（不动） |

---

## 主线判断

| 维度 | 诊断 |
|------|------|
| **主方向** | 山水珍藏型 × 产品美学型（双主方向） |
| **目标客群** | 中年改善 |
| **核心价值** | 江景 + 地段 |

---

## 核心概念

> **「章江归来，一号传世」**

四个词两个动作——章江点江，归来点人，一号点地位，传世点经典。

备选：一江一园一号 / 赣南风物世界归航

---

## 落位方法

- 通感：经典酒店/经典建筑——新古典的传世感
- 动线递进：从大门到阳台，高潮在阳台/院子
- 命名：滨江一号（价值直给型）

## 文化基因

| 文化根 | 对应模块 |
|--------|---------|
| 客家四水归堂 | 会客大厅/下沉庭院/地面拼花 |
| 无风雨归家连廊 | 风雨连廊串联全社区 |
| 赣州宋城文脉 | 建筑立面比例/大门厚重感 |
| 新古典经典比例 | 立面三段式/拱形门洞/对称 |

## 动线递进

| 步骤 | 空间 | 情绪 |
|------|------|------|
| 一 | 城市→大门 | 到了 |
| 二 | 大门→会所 | 好地方 |
| 三 | 大堂→电梯 | 快看到了 |
| 四 | 玄关开门 | 就是这个 |
| 五 ⭐ | 阳台/院子 | 我的 |
