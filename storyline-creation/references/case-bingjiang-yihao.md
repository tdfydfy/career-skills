# 案例：滨江一号 · 赣州

> 完整故事线创作流程，从摸底定向到三件套交付。

---

## 项目信息

| 项 | 内容 |
|----|------|
| 位置 | 赣州章江新区，西侧临章江 |
| 布局 | 西南叠拼洋房（200㎡+） → 东北高层（140㎡+精装），西低东高户户看江 |
| 立面 | 新古典主义，帝国大厦同款风格 |
| 定位 | 高端改善，高出均价30% |
| 阶段 | 设计阶段 |
| 客群 | 40-50岁赣州人，在外打拼（广东/稀土/家具/电商）回乡置业，人生最后一套房 |
| 资源 | 一线临江+滨江公园+章江新区核心最后几块好地 |
| 竞品 | 有江景盘但定位偏低，无成熟故事线 |
| 案名 | 滨江一号（不动） |

---

## 四维纵深扫描

| 纵深 | 判断 |
|------|------|
| **时间** | 章江新区收官阶段——核心区最后几块好地，"错过就没了"的时间点 |
| **空间** | 微观：西低东高一户不遮挡的江景独占。中观：章江新区核心配套成熟。宏观：赣州城市发展主轴 |
| **物性** | 新古典立面+西低东高户户看江+叠拼带院子——产品有"经典"的肉身，老天给了江，人造了经典 |
| **人心** | 40-50岁外出打拼回乡，人生最后一套房。要体面、要犒赏、不想将就。最痛："拼了一辈子，还要将就吗" |

## 交叉锁定

**不可复制的江 × 不可将就的人生 → "章江归来，一号传世"**

---

## 核心主张

> **「章江归来，一号传世」**

四个词两个动作——章江点江，归来点人，一号点地位，传世点经典。
